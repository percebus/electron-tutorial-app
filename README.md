## Setup

### First time

```bash
$> npm run setup
```

### Dependencies

```bash
$> npm install
```

## R&D

### Build

#### `build/`

```bash
$> npm run build
```

#### `out/`

##### `package`

```bash
$> npm run package
```

##### `make`

```bash
$> npm run make
```

### Run

```bash
$> npm start
```
